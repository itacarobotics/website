## Run the Demo

If you haven't already done so in the previous tutorial, clone the tutorial repository:

```bash
git clone https://github.com/itacarobotics/ulixarm_ros2.git
```

Navigate into the repository:

```bash
cd ulixarm_ros2
```

Build the packages:

```bash
colcon build
```

Source the environment:

```bash
source install/setup.bash
```

:::warning
Before launching the demo, **rotate the emergency button** to release it and re-enable motor power.
:::

Launch the demo:

```bash
ros2 launch exercise_2 demo.launch.py
```

RViz will open displaying the robot model along with an interactive marker you can use to set the goal pose.
